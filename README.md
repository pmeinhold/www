### Setup

Using Zola's [Bear theme](https://www.getzola.org/themes/bearblog/) as a Git submodule and adding
[KaTeX](https://katex.org/) by creating `templates/custom_head.html` and adding the
[starter template](https://katex.org/docs/browser) with
[inline math](https://katex.org/docs/autorender#api) to its `<head>`.
